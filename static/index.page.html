"boxes" - the best storage app there is...

...and it's free!

This page will allow you to create boxes, search for the contents, delete boxes and move items around. It is a private instance of the App, which will be enabled for public access once it has been perfected. There are a number of features available in this private instance.

  1. Create a box
  2. Delete a box
  3. Add items to a box
  4. Move items between boxes
  5. Delete items from a box
  6. Search for items in a box
  7. Exit the app
